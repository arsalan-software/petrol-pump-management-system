# Petrol Pump Management System

A Django-based web application for managing petrol pump operations, including shift management, nozzle meter readings, fuel stock, expenses, reports, and customer credit transactions.

## Project Overview

This project was developed to manage daily petrol pump operations in a structured and user-friendly way. It helps the owner record fuel readings, calculate sold liters, track fuel stock, manage expenses, generate daily and monthly reports, and monitor customer credit.

## Key Features

- User login and owner-protected dashboard
- Shift opening and closing workflow
- Nozzle opening and closing meter readings
- Automatic calculation of sold liters and revenue
- Fuel stock and tank management
- Fuel purchase tracking
- Expense management
- Customer credit / Udhaar ledger
- Daily Z-Report
- Monthly Summary report
- CSV export support
- Print-ready reports
- Manual testing of complete business workflow

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
